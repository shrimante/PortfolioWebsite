# Antigravity Portable Setup

This directory contains the "brain" artifacts for this project's localization and development efforts. By committing these files to your repository, you can provide immediate context to Antigravity (or any compatible AI assistant) when working on a different machine.

## Contents
- `brain/task.md`: The current state of the project checklist.
- `brain/implementation_plan*.md`: Technical designs and approved plans.
- `brain/walkthrough.md`: A summary of completed work and verification results.

## How to use on a new machine
When you start a new session with Antigravity on a different machine, you can simply tell the assistant:

> "Please read the files in `.antigravity/brain` to regain context on the current tasks and plans."

The assistant will then be able to resume exactly where we left off.
